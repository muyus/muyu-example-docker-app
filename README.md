# java+mongodb+redis环境搭建

1. 镜像：

  > muyus/muyu-example-docker-java

  > mongo:3.6

  > redis:4.0

  > dockersamples/visualizer:stable

2. 目录：

  > /data/java 存放java相关文件

  > /data/mongo 存放mongo数据

  > /data/redis 存放redis数据

3. 网络：

  > java:9000

  > visualizer:9001

  > mongo:3717

  > redis:7000

4. 内存：

  > java:256m

  > mongo:500m

  > redis:100m
